# Car Dealership Inventory System

A full-stack Car Dealership Inventory System built with Node.js, Express, TypeScript, Prisma, React, and Vite.

## Project Structure

This is a monorepo setup containing both backend and frontend applications.

- `backend/`: Node.js, Express, TypeScript, Prisma
- `frontend/`: React, Vite, TypeScript, TailwindCSS
